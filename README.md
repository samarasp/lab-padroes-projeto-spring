# Explorando Padrões de Projetos na Prática com Java

Repositório com as implementações dos padrões de projeto explorados no Lab "Explorando Padrões de Projetos na Prática com Java". Especificamente, este projeto explorou alguns padrões usando o Spring Framework, são eles:
- Singleton
- Strategy/Repository
- Facade

## Minha entrega

Este projeto foi reproduzido a partir do laboratório da DIO sobre Padrões de Projeto com Java e Spring.

O objetivo foi praticar conceitos como Singleton, Strategy e Facade dentro de uma API REST simples utilizando Spring Boot.

Durante o estudo, observei como o Spring ajuda na organização das camadas da aplicação, separando responsabilidades entre controller, service, repository e model.

### Padrões observados

- **Singleton**: o Spring gerencia os componentes da aplicação como beans, reutilizando instâncias.
- **Strategy**: a interface de service permite separar a regra de negócio da implementação concreta.
- **Facade**: a camada de service simplifica o acesso às operações da aplicação.
